# Trin Portal Web

Portal web sederhana berbasis HTML dan CSS untuk menampilkan halaman informasi dengan tampilan responsif dan navigasi yang mudah digunakan.

## ✨ Features

* Responsive web design
* Simple and clean interface
* Informasi berbasis halaman web
* Navigasi sederhana
* Layout modern dan ringan
* Tampilan mobile-friendly

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

## 🚀 Run Locally

1. Clone repository

```bash
git clone https://github.com/jaqckline/trin-portal-web.git
```

2. Open project folder

3. Jalankan menggunakan:

* Live Server (VS Code)
* atau buka file `index.html`

## 📌 Purpose

Project ini dikembangkan sebagai portal web sederhana untuk pembelajaran dan pengembangan tampilan website berbasis HTML, CSS, dan JavaScript.
